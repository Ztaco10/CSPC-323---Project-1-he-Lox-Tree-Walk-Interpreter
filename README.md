# CSPC-323---Project-1-he-Lox-Tree-Walk-Interpreter

Python implementation of a tree-walk interpreter for the Lox language, following the jlox design pattern from *Crafting Interpreters*.

## Features
- Scanning / lexical analysis
- Recursive descent parsing into an AST
- Expression evaluation
- Variable declarations
- Print statements
- If statements
- While loops
- Block scoping
- Function declarations, arguments, and return values
- Syntax and runtime error reporting

## Run
```bash
python3 lox.py test.lox
